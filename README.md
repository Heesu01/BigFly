# **빅플라이(BigFly)** 
> **빅데이터 분석 자동화 서비스 개발 해커톤 최우수상 수상작**  
> *최적의 공항 출발 시간 추천 및 혼잡도 예측 웹 서비스*
<br/>

## **프로젝트 개요**
![포폴대표사진](https://github.com/user-attachments/assets/88adc263-2570-496a-ac3f-36f0687abdb9)

빅플라이는 공항의 실시간 혼잡도와 항공기 지연 데이터를 기반으로, 사용자에게 최적의 공항 출발 시간을 추천해주는 웹앱 서비스입니다. 출국장의 혼잡 문제와 항공기 지연으로 인한 불편을 최소화하기 위해, 다량의 공공데이터(인천국제공항공사, 한국공항공사 등)와 기상청 데이터를 수집·분석하여 AI 기반 예측 모델로 혼잡도와 운항 상황을 미리 안내합니다. 이 프로젝트는 해커톤에서 높은 평가를 받아 최우수상을 수상했습니다.

<br/>

## **기능 소개**

1. **공항 실시간 혼잡도 조회**  
   - 출국장별 혼잡도를 직관적으로 확인 가능  
   - 시간대별 혼잡도 예측 조회  

2. **맞춤형 공항 출발 시간 추천**  
   - 항공편 정보(출발지, 항공편 시간)를 입력하면 AI 예측 모델을 통해 가장 적절한 공항 도착 시간 안내  
   - 혼잡 시간대를 피하여 불필요한 대기시간을 최소화  
   - 항공기 지연, 결항 가능성 예측  

3. **공항 관련 종합 정보**  
   - 시간대별 출국자 수 예측, 주차장 현황, 공항 뉴스 등 다양한 부가 정보 제공  
   - 여러 공항의 실시간 혼잡도 조회  
   - 기상청 API를 활용한 공항별 기상 상태 연동

<br/>

## **성과 및 기대효과**
- **최우수상 수상** : 높은 완성도를 인정받은 프로젝트  
- **공항 이용 편의성 증대** : 사용자에게 혼잡 시간대 회피를 위한 최적 출발 시간을 제공해 대기 시간을 줄이고 탑승 과정을 원활화  
- **공항 운영 효율성 강화** : 혼잡도 및 운항 현황을 미리 예측해 출국장 인력 배치와 운영 계획에 도움  
- **시간 및 비용 절감** : 승객은 촉박한 탑승으로 인한 불안감을 해소하고, 항공사·공항은 불필요한 지연·인력 낭비 방지

### Branch Naming Convention

| **Prefix** | **Description**          |
| ---------- | ------------------------ |
| `main`     | 서비스 브랜치            |
| `develop`  | 배포 전 작업 기준 브랜치 |
| `feature`  | 기능 단위 구현 브랜치    |

### Commit Convention

| **Prefix** | **Description**                        |
| ---------- | -------------------------------------- |
| `feat`     | 기능 구현, 추가                        |
| `fix`      | 버그 수정, 예외 케이스 대응, 기능 개선 |
| `style`    | UI 디자인 수정                         |
| `api`      | api 연결                               |
| `setting`  | 패키지 설치, 개발 설정                 |
| `refactor` | 코드 리팩터링                          |
| `rename`   | 파일명(폴더명)을 수정                  |
| `remove`   | 파일(폴더) 삭제                        |
| `test`     | 테스트 코드 추가                       |
| `docs`     | README.md 작성 및 변경                 |
| `hotfix`   | 치명적인 버그를 급하게 수정하는 경우   |
